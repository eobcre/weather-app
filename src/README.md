# weather-app

## Run Locally

```
$ git clone https://github.com/eobcre/weather-app.git
$ git cd weather-app
$ npm install
$ npm run dev
```

## Tools

- TypeScript 4.6.4
- React 18.2.0
- react-icons 4.6.0
- Tailwind 3.2.2
- Vite 3.2.3
